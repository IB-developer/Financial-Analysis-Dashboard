# 📊 Financial Performance Dashboard (Microsoft Excel)

## 📌 Project Overview

The **Financial Performance Dashboard** is an interactive Microsoft Excel dashboard designed to analyze and monitor an organization's financial performance. The dashboard compares **Budget vs Actual Spending**, tracks budget variance across multiple business dimensions, and provides actionable insights through interactive visualizations and slicers.

This project demonstrates advanced Excel skills including PivotTables, PivotCharts, calculated fields, slicers, dashboard design, and business analytics.

---

## 🎯 Business Problem

Organizations often struggle to monitor whether departments are spending within their allocated budgets. Financial data is usually scattered across thousands of transactions, making it difficult for management to quickly identify:

- Departments exceeding budgets
- Regional spending trends
- High spending categories
- Quarterly expenditure patterns
- Preferred payment methods
- Overall financial performance

This dashboard consolidates financial data into a single interactive report that supports faster and more informed decision-making.

# 💡 Key Insights

- Actual spending exceeded the planned budget by **11.94%**.
- Over **10,000 financial transactions** were analyzed.
- Marketing and HR recorded the highest percentage of over-budget transactions.
- Travel represented the highest spending category.
- UPI accounted for the largest share of payment transactions.
- Q3 recorded the highest overall expenditure.
